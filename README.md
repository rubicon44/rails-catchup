# 概要
Growは、転職のポートフォリオ用に制作したタスク管理SNSです。

## アピールポイント
- Ruby・Railsによる基本的なCRUDを実装できている
- 開発環境・本番環境にDockerを用いて開発を行っている
- テストをしっかりと記述している
- CI/CDパイプラインを構築している
- AWSで、基本的な本番環境を構築している
